# Медиатека

Локальный менеджер фото и видео на Electron. Приложение хранит файлы рядом с проектом и помогает раскладывать их по категориям, тегам и избранному.

## Быстрый запуск на Windows

Дважды нажмите `Start.bat` в корне проекта. Скрипт сам проверит зависимости и запустит приложение.

Если удобнее PowerShell:

```powershell
.\run.ps1
```

## Ручной запуск

```powershell
npm.cmd install
npm.cmd run start
```

## Где хранятся данные

- Медиафайлы: `programm/data/photos` и `programm/data/videos`
- Метаданные: `programm/metadata.json`
- Миниатюры: подпапки `thumbnails` внутри хранилища фото и видео

## Возможности

- Импорт отдельных файлов или целой папки
- Категории для фото и видео
- Теги, поиск и исключение выбранных тегов
- Избранное, редактирование описания и удаление файлов
- Миниатюры для изображений и видео

Для сборки в отдельный `.exe` можно добавить конфигурацию `electron-builder`.
